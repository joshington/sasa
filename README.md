
Pesasa is a fintech platform that lets you send money to children, employees, 
or beneficiaries — and control exactly how every shilling is spent.

Built for families managing school allowances, companies distributing employee 
benefits, NGOs handling aid disbursement, and the diaspora sending money back home.

→ Parents set spending limits and approve transactions
→ Merchants accept Pesasa payments at point of sale
→ Every shilling is tracked in real time
